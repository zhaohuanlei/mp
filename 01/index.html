小程序需要啥技术？

结构层: html --> wxml 定义了一套新的标记语言

表示层: css --> wxss 部分css3选择器不支持

行为层: js --> 兼容大部分js，并提供了一些API

一个完整项目的流程

接到需求

产品经理：处理需求，并整理文档，制作原型

研发：需求评审；需求合理并且技术可以实现，就继续；否则修改

UI：设计相关界面

前端：写好布局、交互

后端：写接口和文档,前后端联调

测试：提bug，前后端修改bug
